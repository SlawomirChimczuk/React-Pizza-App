# React-Pizza-App
This is simple app where we can search for Pizza recipie 


Starting this app:
1. for React - install vite  - npm create vite@latest

2. for packages - npm install 

3. for server - npm i -D json-server 

For running app: 
npm run dev

App can be viewed on port:
http://localhost:5173/

for running server: 
npm run server

Server can be viewed on port:
http://localhost:5000/
